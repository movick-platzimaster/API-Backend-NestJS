## Features

- <a href="http://nodejs.org" target="blank">Muvick</a> is an on-demand streaming platform.
- Provide entertainment to users at low cost.
- A way to distribute your multimedia content.
- Streaming service with excellent quality and low response times.

## Stack

- NestJS
- MongoDB
- JWT
- Jest

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
